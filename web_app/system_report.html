🤖 AI 기반 상품 임베딩 및 검색 시스템

Google Cloud Vertex AI와 Gemini 모델을 활용한 멀티모달 상품 검색 플랫폼

구축일: 2025년 1월 | 버전: 1.0

📋 프로젝트 개요

본 시스템은 Google Cloud의 최신 AI 기술을 활용하여 상품 정보를 벡터 임베딩으로 변환하고, 텍스트와 이미지를 모두 활용한 멀티모달 검색을 제공하는 웹 애플리케이션입니다.

🎯 프로젝트 목표

  • AI 기반 상품 임베딩 생성 및 저장
  • 텍스트, 이미지, 멀티모달 검색 기능 구현
  • 내부 시스템 검색과 외부 웹 검색 통합
  • AI 이미지 분석을 통한 자동 상품 정보 추출
  • 직관적이고 사용자 친화적인 웹 인터페이스 제공

🏢 적용 분야

  • E-commerce 플랫폼의 상품 검색 시스템
  • 제품 카탈로그 관리 시스템
  • 이미지 기반 상품 추천 시스템
  • 멀티모달 AI 검색 솔루션

🛠️ 설치 및 설정 매뉴얼

1. 시스템 요구사항

필수 요구사항

  • Python 3.12 이상
  • Google Cloud Platform 계정
  • Vertex AI API 활성화
  • Firestore Database 생성
  • Google Custom Search API 키

2. 환경 설정

Google Cloud 설정

# 1. Google Cloud SDK 설치 curl https://sdk.cloud.google.com | bash exec -l $SHELL # 2. 프로젝트 설정 gcloud config set project YOUR-PROJECT-ID gcloud auth application-default login # 3. 필요한 API 활성화 gcloud services enable aiplatform.googleapis.com gcloud services enable firestore.googleapis.com gcloud services enable customsearch.googleapis.com

Python 의존성 설치

# 가상환경 생성 (권장) python -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate # 필수 패키지 설치 pip install flask pandas google-cloud-aiplatform pip install google-cloud-firestore python-dotenv vertexai pip install werkzeug pillow requests numpy

3. 환경 변수 설정

.env 파일 생성

# Google Cloud 설정 PROJECT_ID="your-google-cloud-project-id" LOCATION="asia-northeast3" # Google Custom Search API 설정 GOOGLE_API_KEY="your-google-api-key" CUSTOM_SEARCH_ENGINE_ID="your-custom-search-engine-id"
⚠️ 보안 주의사항:
  • .env 파일을 Git에 커밋하지 마세요
  • API 키는 안전한 곳에 보관하세요
  • 프로덕션 환경에서는 환경 변수로 설정하세요

4. 애플리케이션 실행

웹 서버 시작

# 개발 서버 실행 cd web_app python app.py # 서버 접속: http://localhost:5001

📚 사용자 매뉴얼

1. 상품 등록

단계별 가이드

  1. 상품 등록 페이지 접속: 메뉴에서 "Add Product" 클릭
  2. 상품 정보 입력: 상품명, 설명, 특징 등을 상세히 입력
  3. 이미지 업로드: 상품 이미지 선택 후 "AI 분석하기" 버튼 클릭
  4. AI 분석 결과 활용: 분석된 정보를 텍스트에 추가하거나 교체
  5. 등록 완료: "등록" 버튼을 눌러 상품 저장
💡 팁: 상품 정보를 상세하게 입력할수록 검색 정확도가 향상됩니다.

2. 상품 검색

텍스트 검색

  1. "Product Search" 페이지 접속
  2. "Text Search" 옵션 선택
  3. 자연어로 검색어 입력 (예: "여름용 시원한 티셔츠")
  4. "Search Products" 버튼 클릭

이미지 검색

  1. "Image Search" 옵션 선택
  2. 유사한 상품을 찾고 싶은 이미지 업로드
  3. "AI 분석하기"로 이미지 내용 확인 (선택사항)
  4. "Search Products" 버튼 클릭

멀티모달 검색

  1. "Text + Image" 옵션 선택
  2. 텍스트 설명과 이미지를 모두 입력
  3. 자동으로 이미지 분석이 실행됨
  4. 텍스트와 이미지 정보가 융합된 검색 결과 확인

3. 상품 상세 정보 및 관리

상품 상세 보기

  • 상품 정보: 등록된 모든 상품 정보 확인
  • 이미지 확대: 상품 이미지 클릭으로 확대 보기
  • 기술 정보: 임베딩 차원, 등록 시간 등 확인

유사 상품 검색

  • 내부 시스템 검색: 등록된 상품 중에서 유사 상품 찾기
  • 외부 시스템 검색: Google 검색으로 웹에서 유사 상품 찾기
  • AI 검색어 생성: 상품 정보를 분석하여 자동으로 최적의 검색어 생성

4. 상품 수정 및 삭제

상품 수정

  1. 상품 상세 페이지에서 "수정" 버튼 클릭
  2. 텍스트 정보 수정 또는 새 이미지 업로드
  3. 새 이미지 업로드 시 AI 분석 기능 활용
  4. "수정 완료" 버튼을 눌러 변경사항 저장
⚠️ 주의: 상품 수정 시 새로운 임베딩이 자동으로 생성되어 검색 결과에 즉시 반영됩니다.

🔧 문제 해결 가이드

자주 발생하는 문제

❌ Google Cloud 인증 오류

증상: "Google Cloud credentials not found" 오류

해결방법:

# 1. Google Cloud 인증 확인 gcloud auth application-default login # 2. 환경 변수 설정 확인 export GOOGLE_APPLICATION_CREDENTIALS="path/to/service-account-key.json"

❌ Vertex AI 모델 404 오류

증상: "Publisher Model not found" 오류

해결방법:

  • 프로젝트 ID와 리전 설정 확인
  • Vertex AI API 활성화 확인
  • 모델명 정확성 확인 (gemini-2.5-flash)

❌ Firestore 연결 오류

증상: Database connection failed

해결방법:

  • Firestore 데이터베이스 생성 확인
  • 데이터베이스 ID 설정 확인 ('firestore')
  • 적절한 권한 설정 확인

⚠️ 이미지 업로드 실패

증상: 이미지 업로드 시 오류 발생

해결방법:

  • 지원 형식 확인: PNG, JPG, JPEG, GIF, WebP
  • 파일 크기 제한: 16MB 미만
  • uploads 폴더 권한 확인

로그 확인 방법

# Flask 개발 서버 로그 확인 python app.py # Google Cloud 로그 확인 gcloud logging read "resource.type=vertex_ai_endpoint"
