📋시스템 개요

웹사이트에서 상품 정보를 자동으로 추출하고 AI 임베딩을 생성하여 클라우드에 저장하는 완전 자동화된 배치 시스템입니다.


  • URL 입력만으로 완전 자동 처리
  • Gemini AI를 활용한 정확한 정보 추출
  • Google Cloud 완전 통합
  • 기존 검색 시스템과 100% 호환

🏗️시스템 아키텍처

모듈화된 설계로 각 단계가 독립적으로 작동하며, 실패 시 자동 재시도 및 복구 기능을 제공합니다.


  • 비동기 처리로 높은 성능
  • 지수 백오프 재시도 로직
  • 구조화된 JSON 로깅
  • 설정 기반 동작 제어

🌐웹 크롤링

  • HTML 콘텐츠 자동 추출
  • 속도 제한 및 재시도 로직
  • 다양한 웹사이트 지원

🤖AI 정보 추출

  • Gemini 2.5 Flash 모델 사용
  • 상품명, 설명, 가격 등 자동 추출
  • 구조화된 JSON 출력

🖼️이미지 처리

  • 상품 이미지 자동 다운로드
  • 크기 최적화 및 형식 변환
  • GCS 업로드 및 공개 URL 생성
